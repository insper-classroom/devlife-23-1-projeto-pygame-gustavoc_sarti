GENIUS HEIST

By Gustavo Barroso & João Pedro Sarti

Um desafafio de puzzles, brain teaser e cooperação

Já pensou o quão emocionante deve ser roubar um banco?
Gosta de resolver puzzles e brain teasers?

Então este jogo é para você!

Embarque na aventura de dois ladrões que estão realizando o maior assalto de suas vidas. Será uma jornada cheia de desafios, onde você e o seu parceiro devem trabalhar em equipe e usar a lógica para conseguir fugir com o tesouro sem serem pegôs.

Escolha o personagem que mais agradar e em conjunto enfrentem os desafios de puzzle.

COMO JOGAR

Como baixar?

Para baixar o jogo sera necessario entrar no github e baixar o arquivo ".zip" do jogo. Para isso, basta seguir descendo na pagina ou clicar em 'Jogar' no canto superior direito da tela.

Após baixar o .zip do jogo, extraia para uma pasta de sua preferencia. Agora temos o jogo baixado, mas tambem sera necessario paixar o Python 3 para sua maquina caso ele ja não esteja baixado. O python 3 pode ser encontrado no site oficial do python.

Após ter tudo baixado, basta executar o arquivo 'main.py' que foi extraido. Para isto, utilize o terminal do sistema operacional.


Como jogar?

Este jogo se trata de um jogo de puzzles, sendo assim o personagem tem sua limitação limitada a movimentos absolutos (de um extremo ao outro).
Sabendo disso, utilize as teclas "WASD" para movimentar um personagem para Cima, Esquerda, Baixo e Direita respectivamente. O mesmo vale para o segundo personagem, mas este é controlado pelas setas do teclado.

Segue um guia rapido:
W/↑ (cima)
A/← (esquerda)
S/↓ (baixo)
D/→ (direita)
